# LoadMoreButton

Requires `LoadMoreContextProvider`, with a prop of `onClick`, which is a callback function that takes `LoadMoreDispatch<function>`, `LOAD_MORE_TYPES<array[string]>`
as arguments. See [LoadMoreButton.onClick.js mock](https://github.com/aesop/aesop-gel/tree/develop/src/components/LoadMoreButton/__mocks__/LoadMoreButton.onClick.js)
for an example.
